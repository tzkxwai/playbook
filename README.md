# 🐳 Ansible Docker Automation by sssiqwel

## 📋 Описание проекта
Автоматическая установка Docker и создание контейнеров на удаленных серверах с помощью Ansible.
Проект разработан и протестирован в GitHub Codespaces.

ansible-docker-project/
